Global Basic Income

…地球上の全ての人々が働かなくても生活を送るのに十分なお金が政府から保障される制度

もし、Global Basic Incomeが実現したら、

この世界はどのように変わるだろう？

ゼミ紹介

岡野内ゼミはGlobal Basic Incomeについて研究をしています。
なぜGlobal Basic Incomeなのか。まずはその動機についてお話させていただきます。

みなさんは、この世界で飢餓に苦しんでいる人の割合についてご存知でしょうか？
ＦＡＯ（国連食糧農業機関）による調べでは、世界には、いまだ飢餓に苦しむ人が９人に１人いると言われています。この数を聞いて、多くの方はアフリカなどの発展途上国を思い浮かべたことだと思います。しかし、私たちの生活する日本も例外ではありません。実際、日本は7世帯に１つの世帯が貧困に窮しています。片親世帯となるとその数は2世帯に１つと、一気に貧困率が上がっています。貧困は想像しているよりも、ずっと身近に潜んでいるものなのです。

今回のコロナ騒動で、貧困の危険性を以前よりも身近に感じるようになった方もいると思います。実際、日本では多くの会社が倒産し、多くの人が失業に追い込まれています。授業料を払えなくて退学した学生もたくさんいます。コロナがいまだ収束していないように、今は大丈夫でも数年後の未来の保証はありません。

Global Basic Income
　 このような不安定な世界の救世として、この政策を役立てることはできないでしょうか？

私たち岡野内ゼミは、世界中の人々が幸せになる手段として、Global Basic Incomeが必要だと考えています。

ゼミ講師

活動概要

  • (Global) Basic Incomeについて学ぶ

    • (Global) Basic Incomeの理念を知り、海外の事例などからその実現可能性を探ります。3年生が理解の手助けをしてくれるので、(Global) Basic Incomeのことを知らなくても安心してください！たくさん質問してくださいね!

  • 十大学合同セミナー

    • 毎年4月から7月上旬に行われます。早稲田大学や明治大学など、様々な大学の学生とともに、週２回集まって世界的諸問題について話し合います。活動の最後には30人前後の人数で共同論文執筆をしてもらいます。

    詳しくはこちら
  • ゼミ合宿

    • 夏休み明けぐらいに関西私大との一泊二日の合同合宿があります。合宿までに、各大学で異なる国際問題に関わるテーマの研究を行い、1日目に発表し2日目は合宿地での観光がメインです。

  • 学部研究発表会

    • 毎年11月に、多摩キャンパスのゼミが集まって、1年間のゼミ活動の報告を発表形式で行います。秋学期のゼミはこの学部研究発表会に向けた準備が主な活動内容となります。

  • ゼミ論

    • 冬休み明けの1月には、個々人による論文提出があります。テーマは自由なので、自分の興味のあるテーマについて書くことができます。

  • 海外研修旅行

    • 9月または2～3月に希望者のみが参加する海外へ研修旅行に行きます。
